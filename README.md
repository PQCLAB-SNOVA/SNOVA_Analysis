SNOVA scripts
=============
This repository contains a C program and scripts that have been used to analyze SNOVA.

1. In `scripts`, `rectangular_est.py` gives the security estimates
2. The `minrank` folder contains a  tool to analyze the MinRank in Beullens attack for a given rectangular parameter set with odd $q$.
3. In `sage` the scripts that have been used to find the SNOVA S-matrix are given.
